# ProgrammersCalculator - JavaFX

A programmer's calculator that performs binary arithmetic operations.

## Features
- Binary addition, subtraction, multiplication, division
- Exception handling for invalid inputs and division by zero
- A clean JavaFX GUI

## OOP Principles Demonstrated
- **Encapsulation**: Private fields with getters
- **Abstraction**: Interface and abstract class
- **Inheritance**: Operation classes extend base class
- **Polymorphism**: Used Override Method

## How to Run
1. Clone repository
2. Open in IntelliJ IDEA
3. Run Launcher.java

## Technologies
- Java 21
- JavaFX 21.0.6
- Maven

## Note about Warnings
1. Restricted method warnings - JavaFX is using some low-level system calls that Java might be planning to restrict in future versions 
2. Unsafe warnings - JavaFX graphics rendering uses direct memory allocation for performance
